Mike is an open-source AI legal platform that combines document management, AI-powered chat, tabular data extraction, and automated legal workflows. Built with a Next.js frontend and Express backend, Mike supports Claude and Gemini language models and is designed to be self-hosted with Supabase and S3-compatible storage.

## What Mike does

Mike is purpose-built for legal professionals who need to review, analyze, and extract information from legal documents at scale.

<CardGroup cols={2}>
  <Card title="AI Legal Chat" icon="comments" href="/features/ai-assistant">
    Chat with your documents using Claude or Gemini. The assistant reads your uploaded PDFs and DOCX files, suggests edits with tracked changes, and generates Word documents.
  </Card>
  <Card title="Tabular Reviews" icon="table" href="/features/tabular-reviews">
    Extract structured data from multiple documents simultaneously into a spreadsheet-style grid. Export results to Excel.
  </Card>
  <Card title="Document Management" icon="file-lines" href="/features/documents">
    Upload PDF, DOCX, and DOC files. View full version history with tracked changes, and accept or reject AI-suggested edits.

  <Card title="Legal Workflows" icon="diagram-project" href="/features/workflows">

## Technology stack

Mike is built on a modern open-source stack:

| Layer | Technology |
|---|---|
| Frontend | Next.js 16, React 19, Tailwind CSS |
| Backend | Express.js, TypeScript |
| Database | Supabase (Postgres) |
| Auth | Supabase Auth |
| Storage | S3-compatible (e.g. Cloudflare R2) |
| AI Models | Claude (Anthropic), Gemini (Google) |
| Document conversion | LibreOffice |
